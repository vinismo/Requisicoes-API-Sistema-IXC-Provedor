# Requisições API Sistema IXC Provedor


## Requisições API em PHP, no Sistema do IXC Provedor.

### Método GET
- Listar cadastro de cliente
- Listar cadastro de cliente pelo CPF
- Listar boleto
- Listar contrato
- Listar produto
- Liberação manual

### Método POST
- Inserir boleto
- Inserir cadastro de cliente
- Inserir produto
- Inserir contrato

### Método PUT
- Editar produto
- Editar boleto
- Editar contrato
- Editar cadastro de cliente

### Método DELETE
- Deletar produto
- Deletar boleto
- Deletar contrato
- Deletar cadastro de cliente
